# PhantomVerse Scoreboard

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure `.env` in project root with your contest settings

3. Start development:
   ```bash
   npm run dev
   ```

## Configuration

All settings are in the root `.env` file. Set `VITE_USE_MOCK_DATA=false` for production.
